gwtsa.recharge.recharge module¶

Created on Wed Apr 15 18:20:35 2015

This python script is used to solve the differential equations that are used for the unsaturated zone module. The output of the function is the soil state (how full it is) at each timestep, and the groundwater recharge N.

————————————-Three models can be used: - Percolation Flow: dS/dt = Pe[t] - Kp * (Sr/Srmax)**Gamma - Epu * min(1, Sr/0.5Srmax)

  • Preferential Flow:

dS/ Dt = Pe[t] * (1 - (Sr[t] / Srmax)**Beta)- Epu * min(1, Sr/0.5Srmax)

  • Combination:
dS/ Dt = Pe[t] * (1 - (Sr[t] / Srmax)**Beta) - Kp * (Sr/Srmax)**Gamma - Epu
  • min(1, Sr/0.5*Srmax)

————————————-Numerical info: The Soil module is solved with an implicit euler and Newton Raphson iteration. The initial estimate for the the NR-iteration is provided by an Explicit Euler solution of the above differential equation.
